# Contributing to CoreLift Projects

Thanks for your interest in contributing. Here's how to get started.

## Getting Started

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Submit a pull request

## Guidelines

- Keep PRs focused — one feature or fix per PR
- Write clear commit messages describing the "why"
- Include tests where applicable
- Follow the existing code style in the repository

## Security

If you discover a security issue, **do not open a public issue**. See our [Security Policy](SECURITY.md) for responsible disclosure instructions.

## Questions

Open a discussion or issue in the relevant repository if you're unsure about something before starting work.
